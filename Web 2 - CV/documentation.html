Dokumentation av CV


Vilka problem stötte jag på under projektet?


  • Ett av de största problemen som jag hade var att få positioneringen på olika element att fungera. Jag använde mig av padding, margins och olika former av aligns, men ofta så slutade en bild upp i fel storlek eller positionerad för långt åt ett håll. Detta lyckades jag dock lösa till slut.

  • Ett annat stort problem jag stötte på var när jag råkat lägga in två stycken identiska CSS dokument som jag sedan råkade radera när jag glömt bort att jag redan raderat ett av dem. Detta lyckades jag dock lösa efter att ha sökt genom soptunnan en stund och grävt fram mitt CSS-dokument igen.

  • Detta kanske inte räknas lika mycket som ett problem som resten, men på grund att jag var sjuk en vecka förlorade jag mycket tid och var inte medveten om inlämningsdatumet förräns ett par dagar innan. Alltså hann jag inte lika långt som jag hade velat med projektet.

Dessa var de tre största problemen jag stötte på i mitt projekt.
